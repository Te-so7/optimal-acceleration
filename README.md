# optimal-acceleration
Репозиторий содержит реализацию алгоритма поиска оптимального ускорения для движения материальной точки из начальной позиции в конечную за заданное время. Цель оптимизации — минимизировать затраченную энергию при соблюдении физических ограничений на ускорение и скорость. 
# Оптимальное ускорение на Java (без внешних библиотек)

Этот проект демонстрирует расчёт оптимальной траектории движения с учётом ограничений по скорости и ускорению. Вся логика реализована с использованием только стандартных классов JDK — без подключения сторонних библиотек.

## Особенности проекта

- **Нет внешних зависимостей** — используются только классы из `java.lang`, `java.util` и `java.math`.
- **Простая логика расчёта** траектории с фазами: разгон → постоянная скорость → торможение.
- **Вывод результатов** в консоль в табличном виде.
- **Полностью переносимый код** — работает на любой JVM (Java Virtual Machine).
