# optimal-acceleration
Репозиторий содержит реализацию алгоритма поиска оптимального ускорения для движения материальной точки из начальной позиции в конечную за заданное время. Цель оптимизации — минимизировать затраченную энергию при соблюдении физических ограничений на ускорение и скорость. git clone https://github.com/ваш-аккаунт/optimal-acceleration.git
